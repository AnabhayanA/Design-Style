# Process Notes

## Goal
Build a Material Design web page that illustrates design principles.

## Steps
1. Research Material Design guidelines (material.io)
2. Plan page structure: Hero, Cards, Examples
3. Implement HTML/CSS
4. Add images and interactive buttons
5. Apply elevation and shadow hierarchy
6. Test on desktop and mobile

## Notes
- Primary color: #6200EE
- Secondary color: #03DAC6
- Typography: Roboto, multiple weights
- Motion: 200-300ms hover effects
